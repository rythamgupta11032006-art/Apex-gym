# 💪 APEX GYM

A premium, modern, and fully responsive fitness website built with **HTML, CSS, and JavaScript**. This project demonstrates advanced frontend development techniques, smooth animations, interactive UI components, and responsive web design.

---

## 📖 About The Project

APEX GYM is a high-end gym landing page created as a frontend portfolio project.

The goal was to design a visually appealing website that combines modern aesthetics with smooth user interactions while maintaining clean, scalable code.

---

## 🚀 Features

- Modern Hero Section
- Responsive Navigation
- Animated Statistics Counter
- Smooth Scroll Animations
- Interactive Trainer Cards
- Pricing Toggle (Monthly / Annual)
- BMI Calculator
- Testimonials Carousel
- Contact Form
- Newsletter Subscription
- Back-to-Top Button
- Fully Responsive Design
- Modern Glassmorphism UI
- CSS Animations & Transitions
- Mobile-Friendly Layout

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts
- Intersection Observer API
